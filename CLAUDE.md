## 言語設定

このリポジトリは日本人向け日本語専用プロダクトのため、グローバルルールの言語設定を以下のように上書きする。

- コミットメッセージ：日本語
- README.md：日本語のみ
- CHANGELOG.md：日本語のみ

## バージョン管理

- バージョン記載箇所：`manifest.json` の `version`
- 更新のたびに semver に従ってバージョンを上げること
- 変更内容は以下の2ヶ所に記載すること：

### CHANGELOG.md（全履歴）

- [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) 形式に従うこと（ただし `[Unreleased]` セクションは使わない）
- バージョン見出し形式：`## [1.2.0] - 2026-05-28`
- 変更は `###` 見出しでカテゴリ別に記載すること
  - `追加` / `変更` / `非推奨` / `削除` / `修正` / `セキュリティ`

### README.md（最新バージョンのみ）

- 更新履歴 セクションに**最新バージョンのみ**記載すること
- バージョン見出し形式：`### [1.2.0] - 2026-05-28`
- 変更は `####` 見出しでカテゴリ別に記載すること
  - `追加` / `変更` / `非推奨` / `削除` / `修正` / `セキュリティ`
- 古いバージョンの記載は不要（CHANGELOG.md へのリンクで補完済み）

## コード変更後の必須作業

コードを変更したセッションの終わりには、**必ず以下を行うこと**。ユーザーから言われる前に自分で気づいて対応すること。

1. `manifest.json` の `version` を semver に従って更新
2. `CHANGELOG.md` に変更内容を追記（最新バージョンを先頭に）
3. `README.md` の「更新履歴」セクションを最新バージョンの内容に差し替え
