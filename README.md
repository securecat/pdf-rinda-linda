# PDFリーダーリーダー

視覚障害者など目の不自由な方は、PDFを読むのを諦めざるを得ない状況にあります。それは、世間に現存するPDFのほとんどが、アクセシビリティを考慮していないからです。具体的には、以下のような問題が挙げられます。

- Altエントリが無い
- それどころかテキストレイヤーが無い
- 見た目の文書構造どおりに読み上げられない
- 同一文書構造のPDF文書群であっても、読み上げられ方がばらばらである

本Chrome拡張「PDFリーダーリーダー」は、そうした現実の問題に対処すべく以下の目的遂行のために開発しています。

- どのようなPDFであっても内容を読み解き、見た目の文書構造を再現したMarkdownを出力してスクリーンリーダー等でのPDF読み上げを支援します
- Markdown出力にかかったコストを可視化し、当該PDFの著者に対して、ただPDFを読みたいだけの人たちがどれほどの労力をかけなければ読めないのかという事実を伝え、アクセシブルなPDFづくりを啓発します

## 名称に込めた想い

PDFリーダー界のリーダーを目指し「PDFリーダーリーダー」と名付けました。なお、日本語圏の日本人向けのChrome拡張なので、英語名は特にありませんが、リポジトリ名等で使用するための開発コードネームを「PDF Rinda Linda」にして、そう簡単には世間の同類ソフトウェアたちと被らないようにしました。

## 機能（未実装のものを含みます）

- Chrome内蔵ビューアでPDFを表示している時にテキスト表示用フレームを出して、そこにPDFの内容をMarkdownで出力する
- 出力するMarkdownは以下の内容を用いて構成する
  - テキストレイヤーの内容
  - テキスト的なものをOCRしたテキストの内容
  - 画像等に設定されているAltエントリの内容
  - 画像等をOCRしたテキストの内容
  - 画像等そのものの描画内容をテキスト起こししたもの
- 出力時は可能な限り元のPDFがもっている見た目の文書構造を真似して構成する（見出し構造等）
- テキスト表示用フレーム単体でも音声読み上げが可能
- なお、テキストレイヤーとAltエントリ読み取り以外の機能の実行には、APIキーが必要

## インストール

### Chrome ウェブストア

準備中（当面はデベロッパーモードでの開発のみとします）

### デベロッパーモード（手動インストール）

1. このリポジトリをダウンロードまたはクローン
2. Chromeで `chrome://extensions` を開く
3. 右上の **デベロッパーモード** を有効にする
4. **パッケージ化されていない拡張機能を読み込む** をクリックし、リポジトリのフォルダを選択
