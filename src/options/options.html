PDFリーダーリーダー 設定

Anthropic APIキー

OCR・画像解析・Markdown構造化にはAnthropicのAPIキーが必要です。 キーはお使いのブラウザのローカルストレージにのみ保存され、外部サーバーには送信されません。

APIキーは console.anthropic.com で取得できます。

処理モードについて

機能 APIキーなし APIキーあり
テキストレイヤー抽出 ✓ ✓
画像内テキストOCR — ✓
画像・図版の内容説明 — ✓
Markdown構造化（AI） — ✓
読み上げ ✓ ✓
アクセスコスト表示 ✓ ✓

このツールについて

PDFリーダーリーダーは、テキスト情報の有無に関わらずどんなPDFでも読めるようにするChrome拡張です。 「アクセスコスト」は、そのPDFを読み取るためにかかった処理コストの指標であり、 数値が高いほどPDFのアクセシビリティが低いことを意味します。
